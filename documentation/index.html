Welcome to Studio

Studio is a smartphone simulator for telling interactive narrative stories. This documentation guides you step by step through content creation.

How does it work?

Studio simulates a phone with several applications: Messenger for conversations, InstaPics and OnlySlut for social networks. The player experiences the story through these applications.

As a creator, you don't need to know how to code. Everything is done via text files (.txt) that you simply edit.

Story Structure

Each story is a folder in stories/ with this structure:

  • stories
    • 1-My Story Your story folder
      • desc.txt Description
      • icon.png Icon
      • messenger Conversations
        • characters
        • talks
      • instapics Instagram posts
        • characters
        • posts
      • onlyslut OnlySlut posts
        • characters
        • posts
Tip

To create a new story, simply duplicate an existing story folder and rename it. All necessary files will already be in place.

Basic Principle

Content (InstaPics posts, OnlySlut posts) only appears when it is unlocked via Messenger. The conversation drives the story forward and reveals content to the player.

In Messenger Effect
$talks = next.txt Starts a new conversation
$insta = post.txt Unlocks an InstaPics post + notification
$slut = post.txt Unlocks an OnlySlut post + notification
