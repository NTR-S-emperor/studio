Configuration

Customize your Studio site: translations, music, wallpapers, and more.

Translations

System Text

System text changes (including the site title) can be changed here: system_translations.txt.

Note that this file also allows you to add new languages for translation.

Translating Content

For translations, simply duplicate a .txt file you have already created (whether in Messenger, OnlySlut, or InstaPics) and add the language abbreviation before the name with a "$" prefix.

Original File Translated File (French)
start.txt $FR.start.txt
chapter1/suite.txt chapter1/$FR.suite.txt
Format

For example, "start.txt" and "$FR.start.txt". All you have to do now is apply the translations in the file.

This works for all .txt files: Messenger conversations, InstaPics posts, OnlySlut posts.

Music & Wallpapers

Music

Location: assets/music/

Any MP3 music file from this repertoire will be played randomly.

Random Playback

A song that has already been played cannot be repeated among the next three songs.

Wallpapers

Location: assets/wallpapers/

Any image in this directory will be included in the list of choices for the user's wallpaper.

Girlfriend's Wallpaper (Spy App)

There is a special case for the girlfriend's phone wallpaper. When the player uses the Spy app to view the girlfriend's phone, a specific wallpaper can be displayed.

To set the girlfriend's wallpaper, name the file:

gfwallpaper.png

Place this file in assets/wallpapers/. This image will be used as the background on the girlfriend's phone in the Spy app.

Note

This wallpaper is not available in the player's wallpaper choices. It is exclusively used for the girlfriend's phone in the Spy app.

Site Customization

Favicon

You can change the site's favicon by changing "favicon.png".

Important

You must keep this name and format for the image: favicon.png

Version

The "version.txt" file allows you to change the version number displayed at the top left of the phone to something of your choice.
