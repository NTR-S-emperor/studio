InstaPics

To create posts, you must first ensure that you create your protagonists in the "characters" folder. Posts are then created in the "posts" folder.

Important

As long as a post is not called in a conversation with $insta = post.txt, it will not appear in the application.

Characters

In "characters.txt" you can list different characters that you want to appear on InstaPics.

Sarah (sarah.png)
Lisa (lisa.png)
Robert (robert.png)

To the right of their name, in parentheses, is the avatar image they will use. Be sure to include the file extension. These images should be placed in the "avatar" folder.

Image Format

There is no mandatory format for the image, but a square image is recommended.

Creating Posts

Here, you can create InstaPics posts using .txt files, for example "example.txt".

From the posts folder, you can freely create subfolders, sub-subfolders, etc. You can organize yourself however you like, if there are different paths to your story, etc.

Images Folder

When you create a post and include an image in it, it is important that there is an "images" folder where your .txt file is located and which contains the image for your post.

Post Structure

Sarah
Image: example.jpg
Text: Example text. #example
Likes: 3
Comments
Elvis = That's right!
Ana 5 = Yes
Alan "comments.replied" = I love you
John 18 "replied.replied" = Bastard
Anonymous "replied.replied" = ...
Elena 2 "comments.replied" = It's chat time
Lisa "comments.replied" = I'm stuck.
Anonymous "replied.replied" = Be careful.
Elon = Great!
Line Description
Sarah Name of the character posting the message. This character must have been created in the "characters" folder.
Image: example.jpg Image included in the message. Not mandatory; a post does not necessarily have to be accompanied by a photo.
Text: Example text. Text of the message. Hashtags are automatically formatted.
Likes: 3 Number of 'likes' the message has received.
Comments Start of the comments section.

Comments

Comments go after the "Comments" line, one per line.

Comment Syntax

First the username, then =, then the text of the comment. If you want to add a number of likes to the comment, put the number of your choice to the right of the username.

Example Result
Elvis = That's right! Elvis comments "That's right!" with no likes
Ana 5 = Yes Ana comments "Yes" with 5 likes
Automatic Counter

The people who comment are purely fictional and do not have avatars, so there is no need to create them. The comment counter is automatic and based on the number of comments you have posted.

Replies to Comments

It is also possible to reply to comments, then reply to those replies.

Syntax Description
"comments.replied" Applies to the first comment detected above (to the right of the number of likes, or the username if there are no likes)
"replied.replied" Applies to the line above, so it can respond to a comment or a reply to a comment

Complete Example

Comments
Elvis = Great photo!
Ana 5 "comments.replied" = I agree!
John "replied.replied" = Me too
Marc "replied.replied" = +1
Lisa "comments.replied" = Beautiful
Robert = Top!

Result:

  • Elvis: "Great photo!"
  •     Ana (5 likes): "I agree!" (reply to Elvis)
  •         John: "Me too" (reply to Ana)
  •         Marc: "+1" (reply to John)
  •     Lisa: "Beautiful" (reply to Elvis)
  • Robert: "Top!" (new main comment)

Calling a Post from Messenger

When you call up a post in Messenger with $insta, be sure to specify the path from the "posts" folder.

Example

If you have created an example.txt file in this path:
instapics/posts/lovepath/example.txt

The method for calling it will be: $insta = lovepath/example.txt
