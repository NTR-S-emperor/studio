OnlySlut

To create posts, you must first ensure that you create your protagonists in the "characters" folder. Posts are then created in the "posts" folder.

Same Function as InstaPics

OnlySlut works exactly like InstaPics, except for the calling method: use $slut instead of $insta.

Characters

Characters must be created in onlyslut/characters/.

characters.txt

Sarah (sarah.png)
Lisa (lisa.png)

Avatar images must be in the avatar/ folder.

Creating Posts

Here, you can create OnlySlut posts using .txt files, for example "example.txt".

From the posts folder, you can freely create subfolders, sub-subfolders, etc. You can organize yourself however you like, if there are different paths to your story, etc.

Images Folder

When you create a post and include an image in it, it is important that there is an "images" folder where your .txt file is located and which contains the image for your post.

See InstaPics documentation for full details on post structure and comments.

Calling a Post from Messenger

When you call up a post in Messenger with $slut, be sure to specify the path from the "posts" folder.

Example

If you have created an example.txt file in this path:
onlyslut/posts/lovepath/example.txt

The method for calling it will be: $slut = lovepath/example.txt
