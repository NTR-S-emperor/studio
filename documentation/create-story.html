Create a Story

Naming the Folder

Simply name the folders freely and the story will take the name you have entered.

Format: NUMBER-Story Title

The "NUMBER-" in front of the name is used to position the stories in the order you want them to appear in the menu, and is not displayed in the actual name.

Example

2-A Love Story will be "A Love Story" in the second position in the menu.

You can therefore freely add stories by creating a new folder and following this procedure.

Story Files

desc.txt

The desc.txt file allows you to define the description of your story in the display menu. There is no maximum character limit.

A passionate love story
in a small provincial town.
Your choices will determine the ending.

icon.png

icon.png is the icon that your story will use in the display menu. There is no mandatory format; the icon automatically adapts, but a square format is recommended.

Recommended Method

I recommend that you duplicate the folder from the first story, then rename it correctly to ensure that all the content is there (applications, etc.).
